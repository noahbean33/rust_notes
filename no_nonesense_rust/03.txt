Lesson 1 - First Program, Printing, and Documentation

Watch the video, follow up with exercises. I HIGHLY recommend pausing the video whenever something is not 100% clear, and researching it. This maximizes the learning effect.



Key concepts in this lesson:

println! is a macro (note the !) that prints text to the terminal with a newline.
print! does the same without the newline.
Format placeholders use {} instead of C's %d, %s, etc.
Variables are declared with let. They are immutable by default.

This lesson's source code:

fn main() {
    let a = 10;
    let b = 40;
    let s = "Daniel";
    println!("My first test program!\nThe result is {}\nThis was computed by {}", b / a, s);
}
